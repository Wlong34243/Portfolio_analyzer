# Portfolio Analyzer Pro

A comprehensive portfolio analysis tool that creates detailed Excel reports with:

- Portfolio overview and key metrics
- Technical analysis with buy/sell signals
- Options strategy recommendations
- Risk analysis and concentration warnings
- Multi-account consolidation

## How to Use

1. Upload your broker's portfolio export (CSV or Excel)
2. Adjust analysis settings in the sidebar
3. Review the analysis results
4. Download the complete Excel report

## Features

- 📊 Real-time technical analysis using yfinance
- 💼 Covered call strategy recommendations
- ⚠️ Risk assessment and concentration warnings
- 📈 Support for 5-20 position technical analysis
- 🎯 Customizable analysis parameters
